# 🪞 Reflections

개발자로서의 **실무 경험**, **팀 내 협업**, **개선 활동**, **문화적 고민** 등을 되돌아보고 정리하는 공간입니다.  
기술 그 자체뿐 아니라, **일하는 방식**과 **사람과의 협업**에 대한 생각들을 담았습니다.

> 📌 프로젝트가 끝난 후, 스프린트를 마친 후, 또는 팀에서 새 문화를 도입한 순간들을 돌아보며 정리합니다.

| 파일명 | 설명 |
